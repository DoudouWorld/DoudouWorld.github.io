## 引言

人工智能的核心是算法。从早期的逻辑推理到如今的深度神经网络，AI 算法经历了从符号主义到连接主义、从手工特征到自动表示的演进。本文将系统介绍 AI 领域中几类核心算法的思想、原理和应用场景。

## 一、搜索与优化算法

搜索是 AI 最早使用的方法之一，其核心思想是在一个状态空间中寻找最优解。

### 1.1  uninformed Search（盲目搜索）

**广度优先搜索**（BFS）和**深度优先搜索**（DFS）是最基础的搜索策略：

- **BFS**：逐层扩展节点，保证找到最短路径，但空间复杂度为 O(b^d)
- **DFS**：沿一条路径深入到底，空间复杂度仅 O(d)，但不保证最优解

### 1.2 Informed Search（启发式搜索）

**A* 算法**是启发式搜索的经典代表：

```
f(n) = g(n) + h(n)
```

其中 `g(n)` 是从起点到当前节点的实际代价，`h(n)` 是从当前节点到目标的估计代价（启发函数）。当启发函数是可采纳的（不高估真实代价）时，A* 保证找到最优解。

A* 广泛应用于路径规划、游戏 AI 和机器人导航。

### 1.3 蒙特卡洛树搜索（MCTS）

MCTS 通过随机模拟来评估不同选择的优劣，无需精确的启发函数。它是 AlphaGo 的核心组件之一：

1. **选择**：从根节点出发，沿着最有潜力的路径向下选择
2. **扩展**：在未完全展开的节点上添加新子节点
3. **模拟**：从新节点开始随机对弈至终局
4. **回溯**：将模拟结果沿路径向上更新统计信息

## 二、传统机器学习算法

### 2.1 线性模型

**线性回归**和**逻辑回归**是最基础的监督学习算法：

- **线性回归**：通过最小化均方误差来拟合数据，公式为 `y = w·x + b`
- **逻辑回归**：在线性模型基础上加入 Sigmoid 函数，用于二分类任务

虽然简单，但线性模型具有可解释性强、训练快速等优点，在工业界仍然被广泛使用。

### 2.2 支持向量机（SVM）

SVM 的核心思想是找到一个超平面，使得不同类别的样本到超平面的间隔最大化。对于非线性可分的数据，SVM 使用**核技巧**（Kernel Trick）将数据映射到高维空间：

- **线性核**：`K(x, y) = x·y`
- **多项式核**：`K(x, y) = (x·y + c)^d`
- **RBF 核**：`K(x, y) = exp(-γ||x - y||²)`

SVM 在深度学习兴起之前，是图像分类、文本分类等任务的最强算法之一。

### 2.3 决策树与集成学习

**决策树**通过递归地选择最优特征来划分数据，构建一棵树形结构：

- **ID3**：使用信息增益作为分裂标准
- **C4.5**：使用信息增益率，支持连续属性
- **CART**：使用基尼不纯度，生成二叉树

单个决策树容易过拟合，**集成学习**通过组合多个弱模型来提升性能：

| 方法 | 思想 | 代表算法 |
|------|------|----------|
| Bagging | 并行训练多个模型并投票 | 随机森林 |
| Boosting | 串行训练，每个模型关注前一个的错误 | AdaBoost, GBDT, XGBoost |
| Stacking | 用元模型组合多个基模型的预测 | 模型融合 |

**XGBoost**和**LightGBM**是结构化数据竞赛中的利器，在许多 Kaggle 比赛中占据主导地位。

## 三、神经网络与深度学习

### 3.1 感知机与多层网络

**感知机**（Perceptron）是最简单的神经网络单元：

```
output = activation(w·x + b)
```

多层感知机（MLP）通过堆叠多个感知机层来学习非线性映射。万能近似定理表明：一个包含足够多隐藏单元的 MLP 可以近似任何连续函数。

### 3.2 反向传播算法

反向传播（Backpropagation）是训练神经网络的核心算法，基于链式法则计算损失函数对每个参数的梯度：

```
∂L/∂w = ∂L/∂output · ∂output/∂net · ∂net/∂w
```

配合**随机梯度下降**（SGD）或其变体（Adam、RMSProp），网络可以从数据中自动学习参数。

### 3.3 卷积神经网络（CNN）

CNN 利用**卷积操作**提取空间特征，特别适合处理网格化数据（如图像）：

- **卷积层**：使用可学习的卷积核在输入上滑动，提取局部特征
- **池化层**：降采样，减少参数量，增强平移不变性
- **全连接层**：将提取的特征映射到输出空间

经典 CNN 架构演进：

```
LeNet (1998) → AlexNet (2012) → VGG (2014) → GoogLeNet (2014)
→ ResNet (2015) → EfficientNet (2019) → ConvNeXt (2022)
```

**ResNet**的残差连接解决了深层网络的梯度消失问题，使训练上百层甚至上千层的网络成为可能。

### 3.4 循环神经网络（RNN）

RNN 通过隐藏状态传递信息，能够处理序列数据：

```
h_t = f(W · x_t + U · h_{t-1} + b)
```

但标准 RNN 面临**梯度消失**问题，难以捕捉长距离依赖。**LSTM**和**GRU**通过门控机制缓解了这一问题：

- **遗忘门**：决定丢弃哪些历史信息
- **输入门**：决定保留哪些新信息
- **输出门**：决定当前时刻的输出

### 3.5 Transformer 架构

2017 年，Vaswani 等人在论文《Attention Is All You Need》中提出了 Transformer 架构，彻底改变了 NLP 乃至整个 AI 领域。

Transformer 的核心是**自注意力机制**（Self-Attention）：

```
Attention(Q, K, V) = softmax(QK^T / √d_k) · V
```

其中 Q（Query）、K（Key）、V（Value）是从输入投影得到的矩阵。自注意力允许序列中的每个位置关注其他所有位置，从而捕捉全局依赖关系。

相比 RNN，Transformer 的优势：

- **并行计算**：不依赖序列顺序，可以充分利用 GPU 并行能力
- **全局感受野**：每个位置直接关注所有其他位置
- **可扩展性**：更容易扩展到极大规模

## 四、生成式模型

### 4.1 变分自编码器（VAE）

VAE 结合自编码器和概率推断，学习数据的潜在分布。它由编码器（将输入映射到隐空间分布）和解码器（从隐空间采样重建）组成，通过最大化证据下界（ELBO）来训练。

### 4.2 生成对抗网络（GAN）

GAN 由两个网络组成：

- **生成器**（Generator）：从随机噪声生成逼真样本
- **判别器**（Discriminator）：判断样本是真实数据还是生成数据

两者通过对抗训练不断提升，最终生成器可以产生以假乱真的样本。

GAN 的应用包括图像生成、风格迁移、超分辨率重建等。

### 4.3 扩散模型（Diffusion Model）

扩散模型通过一个逐步加噪和逐步去噪的过程来生成数据：

1. **前向过程**：逐步向数据添加高斯噪声，直到变成纯噪声
2. **反向过程**：训练神经网络学习如何从噪声中逐步恢复数据

DDPM、Stable Diffusion、DALL-E 2 等均基于扩散模型。相比 GAN，扩散模型训练更稳定，生成质量更高，已成为当前图像生成的主流方法。

## 五、强化学习

强化学习（RL）研究智能体如何在环境中采取行动以最大化累积奖励。

### 5.1 马尔可夫决策过程（MDP）

MDP 是强化学习的数学框架，由五元组 `(S, A, P, R, γ)` 定义：

- `S`：状态空间
- `A`：动作空间
- `P`：状态转移概率
- `R`：奖励函数
- `γ`：折扣因子

### 5.2 经典算法

- **Q-Learning**：通过 Q 表学习状态-动作对的价值
- **深度 Q 网络**（DQN）：用神经网络替代 Q 表，解决高维状态空间问题
- **策略梯度**（Policy Gradient）：直接优化策略函数
- **PPO**（Proximal Policy Optimization）：稳定高效的策略梯度算法
- **Actor-Critic**：结合值函数和策略梯度两者的优势

### 5.3 应用

- **AlphaGo**：结合深度神经网络和 MCTS，击败围棋世界冠军
- **机器人控制**：让机器人学会行走、抓取等技能
- **自动驾驶**：决策规划和车辆控制
- **资源调度**：数据中心能量管理、网络资源分配

## 六、大语言模型的核心技术

### 6.1 预训练-微调范式

大语言模型采用两阶段训练：

1. **预训练**：在海量文本上学习语言建模（预测下一个词），获得通用的语言理解能力
2. **微调**：在特定任务数据上进一步训练，适配下游应用

### 6.2 关键训练技术

- **指令微调**（Instruction Tuning）：使用指令-回答格式的数据训练，使模型理解并遵循指令
- **RLHF**（基于人类反馈的强化学习）：利用人类偏好数据训练奖励模型，引导模型生成更符合人类期望的输出
- **思维链**（Chain-of-Thought）：引导模型逐步推理，显著提升复杂推理任务的表现

### 6.3 高效微调方法

面对千亿级参数的模型，全量微调成本过高，出现了多种参数高效微调（PEFT）方法：

- **LoRA**（Low-Rank Adaptation）：在注意力层注入低秩矩阵，仅训练少量参数
- **Prefix Tuning**：在输入序列前添加可学习的前缀向量
- **Adapter**：在网络层中插入小型适配模块

## 总结

AI 算法从最初的符号推理到今天的深度神经网络，经历了几次范式转换。每种算法都有其适用的场景：

| 问题类型 | 推荐算法 |
|----------|----------|
| 结构化数据预测 | XGBoost / LightGBM |
| 图像分类/检测 | CNN（ResNet, EfficientNet） |
| 序列建模（短） | LSTM / GRU |
| NLP / 多模态 | Transformer |
| 图像生成 | 扩散模型 |
| 决策与控制 | 强化学习（PPO, SAC） |
| 对话与创作 | 大语言模型 |

理解这些算法的原理和适用场景，是在实际工作中选择和优化 AI 方案的基础。
